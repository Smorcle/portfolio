## Project Overview

**yek.dev** is a fully custom, full-stack personal portfolio website built to showcase my skills, projects, and contact information as a developer. The site features a retro pixel-art / indie-game aesthetic — inspired by titles like *Celeste* and *Hollow Knight* — with smooth scroll animations, a dark/light mode toggle, a dynamic project feed pulled from a database, and a working contact form.

The goal was to build something that feels hand-crafted rather than template-generated: every section, animation, and visual detail was written from scratch without relying on any CSS or UI frameworks.

## How It Was Built

### 1. Frontend — Single Page Application

The entire site lives in a single `index.html` file divided into four sections: **Hero**, **About**, **Skills**, **Projects**, and **Contact**. Navigation uses anchor links with smooth scrolling.

- **Hero section** features a pixel-art mountain SVG landscape, an animated star field generated in JavaScript, a custom EXP progress bar, and a parallax character sprite.
- **About section** displays animated skill stat bars that fill on scroll using an `IntersectionObserver`.
- **Skills section** groups technologies into four categories (Backend, Frontend, Database, Tools) rendered as pixel-style tag chips.
- **Projects section** is dynamically populated — JavaScript fetches project data from the backend on page load and builds the cards on the client side.
- **Contact section** contains a form that submits via `fetch()` (AJAX) without a full page reload.

A custom CSS cursor and a dark/light mode toggle (persisted in `localStorage`) were also implemented from scratch.

### 2. Backend — PHP + MySQL

The backend is intentionally lightweight, consisting of three PHP scripts:

- **`db.php`** — Establishes a reusable PDO connection to the `my_portfolio` MySQL database with proper error handling.
- **`get_projects.php`** — Queries the `projects` table and returns a JSON response consumed by the frontend.
- **`save_message.php`** — Validates and sanitizes the contact form payload, then inserts it into a `messages` table.

### 3. Admin Panel

`admin.php` (protected by `login.php`) provides a private interface to add, edit, and delete portfolio projects and to read submitted contact messages — removing the need to touch the database directly.

### 4. Styling & Animations

All visual effects are pure CSS3 and Vanilla JS:

- Scroll-triggered `.reveal` class animations use `IntersectionObserver`.
- The star field is procedurally generated by `script.js` on page load.
- Dark mode switches a single `data-theme` attribute on `<html>`, and CSS custom properties (`var(--bg)`, `var(--accent)`, etc.) cascade the change instantly across the entire page.
- Typography uses *Press Start 2P* for headings (reinforcing the retro-game feel) and *Inconsolata* for body text (a developer-friendly monospace font).

## How to Run Locally

1. Clone the repository: `git clone https://github.com/Smorcle/portfolio.git`
2. Place the folder in your web server's root directory (e.g., `htdocs` for XAMPP).
3. Import the SQL schema into phpMyAdmin under the database name `my_portfolio`.
4. Start Apache and MySQL, then open `http://localhost/portfolio` in your browser.
